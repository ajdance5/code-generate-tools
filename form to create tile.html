Create Deals Tile

This tool will generate the code for deals landing page tiles for Chrysler Jeep Dodge and Fiat Alfa

Notes:

  • You do not have to strip out anything from the image src url (eg: 340/auto/maxfit), however it will still work if you do
  • The 'Heading' field has a character limit of 40 characters
  • The 'Text' field had a character limit of 85 characters
  • The link field automatically creates and internal link. If you need your deals tile to link off page you will need to update the code manually
